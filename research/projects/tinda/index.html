TINDA - assess your digital learning competences 

TINDA is an online self-assessment tool for evaluating digital competences and achieved qualification requirements for teachers in all educational levels.

Technical specification

Tinda is developed using PHP content management framework Drupal. The current version of Tinda is running on Drupal 8.6.

It has been customized using following modules developed for Tinda:

  • Questions: To import questions from Tao (a question authoring platform) to Tinda.
  • Qualification Test: To create qualification test.
  • Qualification test round: To create a test round from qualification test, assign it to users, and specify the deadline.
  • Submission: To store the submitted answers for each test round.
  • CSV Downloader: To download data from all submissions in CSV format.
